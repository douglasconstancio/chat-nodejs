## ⬇️ Como baixar

```bash
    // Clonar repositório
    $ git clone https://github.com/douglasconstancio/chat-nodejs.git

    // Acessar diretório
    $ cd chat-nodejs

    // Instalar dependências
    $ yarn

    // Realizar as migrações
    $ yarn migration:run

    // Iniciar projeto
    $ yarn dev
```
